# IPL Analysis Dashboard 🏏

## Overview
Comprehensive IPL (Indian Premier League) cricket data analysis dashboard built with Power BI, providing insights into team performance, player statistics, and match outcomes.

## Dashboard Features

### Key Metrics Displayed:
- **Title Winner**: Mumbai Indians
- **Orange Cap Winner**: DA Warner (692 Runs)
- **Purple Cap Winner**: Imran Tahir (52 Wickets)
- **Tournament Statistics**: 786 matches, 1654 total games

### Visualizations Include:
1. **IPL Batting Stats**: Total runs, strike rate, boundaries analysis
2. **IPL Bowling Stats**: Wickets, economy rate, bowling strike rate
3. **Venue Analysis**: Matches won by different venues
4. **Result Type Analysis**: Win by wickets vs runs vs super over
5. **Toss Decision Impact**: Win/loss based on toss decisions

## Tools & Technologies
- **Power BI Desktop**: Dashboard creation and visualization
- **Data Source**: IPL match and player statistics
- **Analysis Period**: Multiple IPL seasons

## Key Insights
- Mumbai Indians emerged as the most successful team
- Venue-wise performance analysis shows clear home advantages
- Toss decisions significantly impact match outcomes
- Batting vs bowling performance metrics reveal team strengths
